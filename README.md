# 🌦️ Python Weather App

A desktop weather application built using **Python** and **Tkinter**. The app connects to the **OpenWeatherMap API** to fetch real-time weather information based on a user's chosen city.

## ✨ Features

- 🌡️ Display current temperature in Celsius
  
- ☁️ Show current weather conditions
  
- 🌍 Search weather by city name
  
- 💨 Display wind speed
  
- 💧 Show humidity levels
  
- 📈 Display atmospheric pressure
  
- 🔼 Show maximum temperature
  
- 🔽 Show minimum temperature
  
- 🌅 Display sunrise and sunset times

## 🛠️ Technologies Used

- 🐍 **Python**
  
- 🖥️ **Tkinter** - Graphical User Interface
  
- 🌐 **OpenWeatherMap API** - Weather data
  
- 📦 **Requests Library** - API requests
  
- 📄 **JSON** - Data processing

## ⚙️ How It Works

1. 👤 User enters a city name into the application.

2. 🌐 The app sends a request to the OpenWeatherMap API.
  
3. 📥 The API returns weather data in JSON format.
   
4. 🔍 Python processes the data.
   
5. 🖥️ The weather information is displayed in the Tkinter interface.

## ▶️ How to Run

1. Download the repository.

2. Open the project folder in **Visual Studio Code**.

3. Open `weather.py`.

4. Press the ▶️ **Run Python File** button in the top-right corner.

5. The Weather App will launch. 🌦️
